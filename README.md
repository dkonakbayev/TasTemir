# Fitness Booking App

**Project Description**  
Fitness Booking App is an application for booking fitness classes. The platform allows fitness centers to register, create, and manage classes, while users can browse schedules and book sessions.  

The project is divided into frontend and backend:  
- **Frontend:** Angular  
- **Backend:** Django REST Framework  

---

## Features

### For Fitness Centers:
- Register and manage profile  
- Add and edit classes  
- View list of users registered for classes  

### For Users:
- Browse class schedules  
- Book classes  
- Cancel bookings  

---

## Technologies

- **Frontend:** Angular, TypeScript, HTML, CSS  
- **Backend:** Django REST Framework, Python  
- **Database:** PostgreSQL  
- **Authentication:** JWT
